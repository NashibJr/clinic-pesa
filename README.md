# Clinic Pesa - Home Dashboard Assignment

This is the technical assignment given to me by Clinic Pesa, implementing the home dashboard UI using React + TypeScript + Vite.

## Features

- Top bar with app name and greeting
- Savings card with balance, frequency, autosave amount, and eye icon
- Tabs: Account Options, Invest, Transactions
- Four action cards:
  - Pay Medical Bill
  - Get Loan
  - Join MaMas
  - My Approvals
- Mobile-first responsive design

## Libraries Used

- **React** - Frontend library
- **TypeScript** - Type safety
- **Vite** - Fast dev server & build tool
- **Tailwind CSS** - Styling & responsiveness
- **React Icons** - Icons for cards and top bar
- **Swiperjs** - Carousel

## Getting Started

### Install dependencies

`bash`
npm install

# or

### Running the application
